# 📰 新闻爬取器

基于 [ScriptCat](https://docs.scriptcat.org/) 的新闻 RSS 爬取与导出工具。

## 功能特性

- 🔄 **RSS 订阅源解析** - 支持 XML (RSS/Atom) 和 JSON 格式
- 💾 **本地数据存储** - 新闻保存在浏览器本地，跨标签页同步
- 📥 **Markdown 导出** - 一键导出新闻为 `.md` 文件
- 🎨 **轻量浅色浮窗** - 不碍眼的浅色主题新闻面板
- ⚙️ **自定义新闻源** - 添加/删除/重置新闻源
- 📱 **右键菜单入口** - 通过脚本猫图标菜单快速操作

## 安装前提

- 安装 [ScriptCat](https://docs.scriptcat.org/) 浏览器扩展
- 启用脚本猫的脚本引擎

## 安装使用

1. 复制 `news-crawler.user.js` 的全部内容
2. 打开 ScriptCat → 编辑器 → 新建脚本
3. 粘贴代码并保存
4. 在目标页面点击脚本猫图标使用

## 使用方法

### 菜单选项

| 菜单 | 功能 |
|------|------|
| 📰 查看新闻列表 | 打开/关闭新闻浮窗 |
| 🔄 刷新新闻 | 重新抓取所有新闻源 |
| 📥 导出为 Markdown | 下载新闻为 `.md` 文件 |
| ⚙️ 配置新闻源 | 管理添加新闻源 |
| 🗑️ 清空新闻数据 | 清除本地存储的新闻 |

### 新闻浮窗

- 位置：页面右下角
- 操作：点击新闻条目用新标签打开原文
- 工具栏：刷新 / 导出按钮
- 点击 ✕ 或再次点击「查看新闻列表」关闭

### 配置新闻源

1. 点击 ⚙️ 配置新闻源
2. 填写名称、RSS URL、类型（xml/json）
3. 点击「添加」
4. 支持删除现有新闻源和恢复默认

  > 新闻摘要内容...

- [新闻标题](https://example.com) - 2026-04-18

## 技术实现

| API | 用途 |
|-----|------|
| `GM_xmlhttpRequest` | 跨域请求 RSS |
| `GM_setValue/getValue` | 数据持久化存储 |
| `GM_download` | 文件导出下载 |
| `GM_addElement/addStyle` | UI 浮窗构建 |
| `GM_openInTab` | 新标签打开链接 |
| `GM_notification` | 操作通知提示 |

## 常见问题

**Q: 某些新闻源抓取失败？**
A: 部分网站有访问限制或 CORS 限制，可尝试添加其他可用的 RSS 源。

**Q: 如何添加自己的新闻源？**
A: 点击「⚙️ 配置新闻源」→「添加新闻源」，填写名称和 RSS URL。

**Q: 数据存储在哪里？**
A: 数据存储在浏览器扩展的本地存储中，可通过脚本猫设置查看。

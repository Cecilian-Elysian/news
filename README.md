# 📰 新闻爬取器

一键抓取新闻、自动生成日报并导出的浏览器扩展脚本。

## 功能特点

- 📰 **一键抓取** - 同时从多个新闻源获取新闻
- 📑 **自动生成日报** - 整理成 Markdown 格式的每日报告
- 🌙 **夜间模式** - 保护眼睛的深色主题
- 📁 **自定义导出位置** - 可选择下载文件夹直接保存
- ⚙️ **灵活配置** - 添加/管理新闻源，调整透明度
- 🔄 **智能回退** - 抓取失败时自动尝试其他方式

## 支持的新闻源类型

| 类型 | 说明 |
|------|------|
| RSS | RSS 订阅源 |
| JSON | JSON API 接口 |
| 网页 | 直接解析网页提取新闻 |

## 默认新闻源

- 新浪新闻、腾讯新闻、网易新闻、搜狐新闻
- 36氪、虎嗅、IT之家、观察者网
- 澎湃新闻、知乎热榜、百度热搜
- Bilibili、GitHub趋势、HackerNews
- RadarAI 等 20+ 新闻源

## 安装使用

### 安装方式

1. 安装 ScriptCat 或 Tampermonkey 浏览器扩展
2. 复制 `news-crawler.user.js` 内容到扩展中创建新脚本
3. 刷新任意网页即可使用

### 基本操作

1. 点击右下角 📰 悬浮按钮打开侧边栏
2. 点击「🚀 一键抓取并生成日报」获取新闻并导出
3. 设置中选择下载文件夹方便后续自动保存

## 界面预览

- 右侧悬浮窗设计，不影响正常浏览
- 统计面板显示新闻条数、来源数量等
- 新闻按来源分组展示，点击可打开原文

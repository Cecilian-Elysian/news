# 📰 新闻爬取器

一键抓取新闻、自动生成日报并导出的浏览器扩展脚本。

## 功能特点

- 📰 **一键抓取** - 同时从多个新闻源获取新闻，支持并行抓取
- 📑 **自动生成日报** - 整理成 Markdown 格式的每日报告
- 🌙 **夜间模式** - 保护眼睛的深色主题
- 📁 **自定义导出位置** - 可选择下载文件夹直接保存
- ⚙️ **灵活配置** - 添加/管理新闻源，调整透明度
- 🔄 **智能回退** - 抓取失败时自动尝试 RSS → JSON → 网页 等多种解析方式
- 📊 **抓取状态** - 实时显示每个新闻源的抓取状态和错误信息
- 🩺 **健康检测** - 一键检测所有新闻源可用性

## 支持的新闻源类型

| 类型 | 说明 |
|------|------|
| RSS | RSS 订阅源 |
| JSON | JSON API 接口 |
| 网页 | 直接解析网页提取新闻 |

## 安装使用

### 安装方式

1. 安装 ScriptCat 或 Tampermonkey 浏览器扩展
2. 复制 `news-crawler.user.js` 内容到扩展中创建新脚本
3. 刷新任意网页即可使用

### 基本操作

1. 点击右下角 📰 悬浮按钮打开侧边栏
2. 点击「🚀 一键抓取并生成日报」获取新闻并导出
3. 设置中选择下载文件夹方便后续自动保存

## 界面预览

- 右侧悬浮窗设计，不影响正常浏览
- 统计面板显示新闻条数、来源数量等
- 新闻按来源分组展示，点击可打开原文
- 抓取状态实时显示成功/失败数量

|------|------|
| 仅标题 | 快速生成，只包含新闻标题和链接 |
| 含摘要 | 获取文章摘要，提供更完整的信息 |

## 注意事项

- 部分新闻源可能需要网络代理才能访问
- 首次使用建议选择「仅抓取新闻」测试各源是否正常
- 导出 Markdown 文件可用 VS Code、Typora 等工具阅读
- 使用 `rsshub.app` 聚合源时需要稳定的网络连接
